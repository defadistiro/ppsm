Cara Setup — 3 Langkah
Langkah 1 — Deploy GAS:
Buka script.google.com, buat project baru, paste kode GAS v4.2, lalu:
Deploy → New Deployment → Web App → Execute as: Me → Who has access: Anyone → Deploy

Langkah 2 — Izinkan akses Drive:
Saat pertama deploy, Google minta izin akses Drive — klik Allow. Jika ada popup "does not have permission to call UrlFetchApp", berarti perlu jalankan fungsi testKoneksi() sekali dari editor GAS untuk grant permission.

Langkah 3 — Isi URL di bawah:
Salin URL deployment (format: https://script.google.com/macros/s/…/exec) ke kolom URL Apps Script di bawah, lalu klik 💾 Simpan.
⚠️ Penyebab error "does not have permission to call UrlFetchApp.fetch":
GAS belum pernah dijalankan manual. Buka GAS editor → pilih fungsi testKoneksi → klik Run → setujui semua permission yang diminta.
